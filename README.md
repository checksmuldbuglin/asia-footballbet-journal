# Asia FootballBet Journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要围绕足球赛事相关的数据记录与信息展示，不依赖任何特定域名或网站。

## 项目简介

本仓库作为一个静态页面集合，旨在以简洁的 HTML 格式存放各类足球相关数据页面。每个页面均为独立文件，便于浏览、存档与分发。

## 目录结构

```
asia-footballbet-journal/
├── index.html          # 主页入口
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、图片等）
│   ├── styles.css
│   └── images/
└── README.md           # 本文件
```

仓库根目录下的 `index.html` 可作为导航入口，`pages/` 文件夹内为具体的内容页面。`assets/` 存放页面共用的样式与素材文件。

## 页面归档说明

- 所有页面均为纯 HTML 文件，不依赖外部服务器端环境。
- 页面之间相互独立，可单独打开或通过主页链接跳转。
- 内容更新时，新增或修改对应文件即可，无需改动其他页面。
- 历史版本可通过 Git 提交记录追溯。

## 维护说明

- 欢迎通过 Issue 提出建议或报告问题。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 请保持页面风格统一，公共样式建议引用 `assets/styles.css`。
- 避免在页面中添加指向外部未知来源的链接或资源，以保证页面独立性。
